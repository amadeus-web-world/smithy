## Steps to Export

DIV-LARGELISTWITHITEMSEPARATOR

1. go to the [Gemini AI](https://gemini.google.com/) conversation you have concluded.
2. Make sure the [Chrome Extension ai-chat-exporter.com](https://chromewebstore.google.com/detail/jfepajhaapfonhhfjmamediilplchakk) is enabled and that this toolbox next to send message is visible.
> ![export-select](%node-assets%google-gemini/export-select.png)
3. Scroll right to the top of the conversation - this can be a painstaking process but hang in there :)
4. Click Select > All. Screen may take a minute to refresh
5. Then click Export > To Markdown
> ![export-to-markdown](%node-assets%google-gemini/export-to-markdown.png)
6. File should get saved to "Downloads Folder"

## Steps to Publish

DIV-LARGELISTWITHITEMSEPARATOR

1. Move to WithAI folder for example
> D:\AmadeusWeb\alliances\people\imran\2025\2025-12\11-to-15
2. Move exported file from downloads here
3. Rename file to serial no -- file - name with NO spaces and hyphens between words. Always letters should be in lower case.
4. Add Html Comment Marker like so<br />
> <textarea rows="3" style="width: 100%;" readonly="readonly">&lt;!--exported-from-gemini-ai--&gt; CRLF
</textarea>
5. Navigate to the link. Note convention is to have a parent folder per 5 (like 11-to-15)
6. Check that encoding is correct, UTF icons are loading AND so are outputted tables.
